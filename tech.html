Technology

Our Technology division teaches a variety of building skills, from using power tools to learning the mechanics behind electronic vehicles. All of our events are hands-on, which provides opportunities to learn how to build machines and structures.

We're composed of 6 events: Bridges, Code Busters, Gravity Vehicle, Ping-Pong Parachute, Detector Building, and Wright Stuff.

Code Busters

This event involves cryptography - students decipher various types of codes, ranging from mono-alphabetic substitution to hill ciphers.

  • Learn More

Bridges

This event tests one's skills to design and build a bridge which is very lightweight and capable of holding a designated weight.

  • Learn More

Gravity Vehicle

This event requires members to build the fastest and most accurate vehicle possible using a ramp mechanism.

  • Learn More

Ping-Pong Parachute

This event requires members to build a rocket and parachute to launch and keep a ping pong ball in the air for as long as possible.

  • Learn More

Detector Building

This event requires one to build a durable Conductivity Device that will accurately measure and display both voltage and concentrations of NaCl.

  • Learn More

Wright Stuff

This event challenges you to build a rubber-powered wooden plane that can stay flying in the air for as long as possible.
